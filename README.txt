To start the web server-
Type 'java SimpleWebServer [port] [path]'
E.g. 'java SimpleWebServer 8888 ./public'

To recreate the error-
404: in the browser go to localhost:xxxx/dfsdfs.png, or any file that isn't present in the public folder.
500 and 302: in the browser go to: localhost:xxxx /redirect
